# 🛡️ Social Engineering Awareness Simulator

A web-based cybersecurity awareness platform built using **Python, Flask, SQLite, HTML, CSS, JavaScript, and Bootstrap**. The project educates users about common social engineering attacks through interactive simulations, quizzes, analytics, and security awareness modules.

---

## 📌 Project Overview

Social Engineering Awareness Simulator is designed to help users identify and defend against various social engineering attacks by providing realistic attack simulations in a safe learning environment.

The platform allows users to experience phishing attempts, fake login pages, malicious URLs, SMS scams, QR code attacks, USB attacks, and more while tracking their performance through an analytics dashboard.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/sachinsingh12401390/Social-Engineering-Awareness-Simulator.git
```

Move into the project folder

```bash
cd Social-Engineering-Awareness-Simulator
```

---

## Create Virtual Environment

Windows

```bash
python -m venv venv
```

Activate

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run Application

```bash
python app.py
```

Open browser

```
http://127.0.0.1:5000
```
